# Decentralized Blog on Stellar (Soroban)

## 📌 Project Description

This project is a decentralized blogging platform built using Soroban smart contracts on the Stellar network. It allows users to create and store blog posts on-chain without relying on centralized servers.

The goal of this project is to demonstrate how decentralized storage and user authentication can be implemented using Soroban.

---

## ⚙️ What It Does

- Users can create blog posts using their Stellar address.
- Each post is stored on-chain with a unique ID.
- Anyone can read posts directly from the blockchain.
- The contract maintains a global post counter.

---

## 🚀 Features

- ✍️ Create blog posts (title + content)
- 🔐 Author authentication using Stellar addresses
- 📖 Read individual posts by ID
- 📚 Fetch all posts from the contract
- 🔢 Automatic post ID generation

---

## 📦 Tech Stack

- Soroban (Stellar Smart Contracts)
- Rust
- Stellar SDK

## ⚠️ Limitations

- No editing or deletion of posts
- No pagination (may become inefficient with many posts)
- No user-specific post indexing
- No comments or interactions

---

## 🔮 Future Improvements

- Implement pagination for scalability
